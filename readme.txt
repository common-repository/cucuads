This Monetize plugin offers ads for your mobile traffic

== Description ==

CucuAds is the best way to <strong>monetize your mobile traffic</strong> in Wordpress. It is extremely easy to install obtaining a high revenue while being fully customizable to your needs. To start just <a href="https://cucuads.com/" target="_blank">Sign up</a> to get your own ID, and then go to your configuration page within your Wordpress configuration to save it.

== Installation ==

Upload the CucuAds plugin to your blog, activate it.
Follow the instructions from description to properly configure it.
